void CheatActivateChetkido ( )

Activates the Chetkido cheat code.

Decrypts the string "The password is: Chetkido" and displays it on the screen if the following conditions are met:

  • The level is snow_approach
  • The game has 100% completion
  • Coin count is 99
  • Life count is 0

Unofficial name because the real name is unknown.
